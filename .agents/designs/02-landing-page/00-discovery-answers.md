# Discovery Answers — Landing Page

## Product & Users
- **Target audience**: Polish university students at Merito Group (14 universities), aged 18–26, tech-comfortable, use Duolingo / LinkedIn / Notion daily
- **Product type**: Marketing landing page → conversion to onboarding
- **Key emotion**: Motivated + Confident ("I know exactly what I'm missing and how to fix it")
- **Competitor references**: Duolingo (gamified learning), LinkedIn (professional credentialing), Coursera (edtech courses)

## Visual Direction
- **Style archetype**: 🌅 Cosmic Campus / Hybrid — light main content + dark hero/footer + indigo–cyan gradient + emerald CTA
- **Color preference**: Indigo + Cyan as primary gradient; Emerald for CTA (growth/success)
- **Typography**: Nunito (display, friendly + student-facing) + Inter (body) + JetBrains Mono (stats/numbers)
- **Layout density**: Spacious — generous sections, clear hierarchy
- **Animations**: Subtle + smooth (hover lifts, fade-ins, blob float)
- **Microcopy tone**: Professional Polish + English tech terms (e.g., "Market Intelligence", "Skill Gap")
- **Dark mode**: Partial — dark hero + dark footer + dark header (visual frame), white body

## Scope
- **Screens**: 1 screen (landing page / marketing page)
- **PRD**: Plan 02 from `.agents/plans/02-landing-page.md`
- **Existing assets**: None — new brand identity
- **Priority**: Conversion to /onboarding CTA
